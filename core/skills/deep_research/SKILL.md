\## Research Standards



\*\*Good Research:\*\*

```text

// ‚úÖ GOOD: Multi-source verification \& Synthesis

"While Report A \[1] claims X, Study B \[2] suggests this only applies in context Y. Therefore, X is likely true when..."



// ‚úÖ GOOD: Handling uncertainty \& Nuance

"There is no consensus on X. The leading theory \[1] suggests A, while a newer hypothesis \[2] proposes B."



// ‚úÖ GOOD: Primary sources preference

"The official API documentation \[1] states..." (Preferred over "A blog post said...")

```



\*\*Bad Research:\*\*

```text

// ‚ùå BAD: Reliance on single unverified source

"This blog post says X is true."



// ‚ùå BAD: Hallucination / Assumption

"It is likely that they use this technology." (without evidence)



// ‚ùå BAD: Surface-level summary

"I found 5 articles that all say the same thing." (Failure to dig deeper for dissenting views)

```
