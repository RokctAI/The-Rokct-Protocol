# Extensions Registry

This directory contains **Pluggable Capabilities** for the Rokct Protocol.
Just like VS Code Extensions, these add specific "Skills" or "Configs" to the Agent.

## Directory Structure

### [Prompts](./prompts/)
Reusable, high-quality instructions for specific tasks.
*   *Usage*: "Run prompt: Security Audit"
*   *Example*: `security_audit.md`, `code_review.md`

### [Schemas](./schemas/)
Strict output formats (JSON/Markdown) to ensure consistency.
*   *Usage*: "Use the Feature Plan Schema"
*   *Example*: `coding_schemas.md`

### [Skills](./skills/) (Future)
Complex workflows requiring multiple steps or external tools.
